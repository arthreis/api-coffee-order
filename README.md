## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker
$ docker-compose up

# watch mode
$ npm run start:local

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
